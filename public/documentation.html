🏫 Web ENT API Authentification Docs Téléchargement

web-ent-api.xyz/api/version

Renvoie la version de l'api


web-ent-api.xyz/api/getevaluation/:token

Renvoie la moyenne générale de l'utilisateur


web-ent-api.xyz/api/gethomework/:token/:date/:twodate/:nolink

Renvoie les devoirs pour la date précisée et le nombre de devoirs. Twodate : si false renvoie les devoirs justes pour la première date sinon les devoirs pour les deux dates comprises. nolink si true enlève les liens ou lien vers un fichier entre crochets si false ne les enlève pas. (les dates doivent être au format : yyyy-MM-dd Ex : 2023-10-25)


web-ent-api.xyz/api/getagenda/:token/:date/:twodate

Renvoie l'agenda pour la date précisée et le nombre de cours. Twodate : si false renvoie l'agenda juste pour la première date sinon l'agenda pour les deux dates comprises. (les dates doivent être au format : yyyy-MM-dd Ex : 2023-10-25)


web-ent-api.xyz/api/getperiodid/:token

Renvoie de l'Id de period pour getevaluation


web-ent-api.xyz/api/getevaluation/:token/:average

Renvoie la moyenne générale de l'utilisateur si la réponse est : "Nan" le period Id n'est pas valide. average si true renvoie la moyenne générale de l'utilisateur sinon la moyenne de toutes les matières où l'utilisateur à des notes. S'il n'a pas de notes dans cette matière il ne renverra ni la moyenne ni le professeur
